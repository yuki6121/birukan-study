## 1. 行政概論：期間・頻度マトリクス
試験に出る「期間」は限られています。これらを完璧に暗記します。

### 1-1. 届出・報告
| 項目 | 期間・期限 | 提出先 | 備考 |
| :--- | :--- | :--- | :--- |
| 特定建築物の届出（使用開始） | 使用開始日から **1ヶ月以内** | 都道府県知事 | |
| 届出事項の変更 | 変更日から **1ヶ月以内** | 都道府県知事 | |
| 特定建築物の非該当届 | 非該当となった日から **1ヶ月以内** | 都道府県知事 | 「30日」ではなく「1ヶ月」 |
| 管理技術者の選任 | **直ちに** (遅滞なく) | - | 欠員が出た場合も同様 |

### 1-2. 検査・点検頻度（建築物環境衛生管理基準）
| 対象設備・項目 | 頻度 | 具体的内容・備考 |
| :--- | :--- | :--- |
| **残留塩素測定** | **7日以内**ごとに1回 | 遊離0.1ppm以上 (結合0.4ppm以上) |
| **空気環境測定** | **2ヶ月以内**ごとに1回 | 温度・湿度・気流・CO・CO2・粉じん・HCHO |
| **飲料水の水質検査** | **6ヶ月以内**ごとに1回 | 11項目（一般細菌, 大腸菌等） |
| (水質検査・省略不可) | (毎日) | 色、濁り、臭い、味 |
| **給水栓の清掃** | **6ヶ月以内**ごとに1回 | 貯水槽清掃は1年以内ごと |
| **排水槽(ビルピット)清掃** | **6ヶ月以内**ごとに1回 | |
| **ねずみ・昆虫等調査** | **6ヶ月以内**ごとに1回 | 生息状況の点検（統一的実施） |
| **冷却塔の清掃** | **1年以内**ごとに1回 | 使用期間前・使用期間中(月1回) |
| **加湿装置の清掃** | **1年以内**ごとに1回 | 使用期間前・使用期間中 |
| **建築物定期調査報告** | (特定行政庁が定める) | おおむね6ヶ月〜3年ごと |

### 1-3. 保存期間
| 書類の種類 | 保存期間 | 備考 |
| :--- | :--- | :--- |
| 管理基準に基づく帳簿類 | **5年間** | 測定結果、点検記録など |

| :--- | :--- | :--- |

## 3. 計算公式集 (物理・空調・給排水)
公式は「形」を覚えて、数値を当てはめる練習をすること。

### 3-1. 換気と濃度
**必要換気量 $Q$ (m³/h)**
$$Q = \frac{M}{C - C_0}$$
* $M$: 汚染物質発生量 (mg/h, m³/h)
* $C$: 室内許容濃度 (mg/m³, ppm)
* $C_0$: 外気濃度 (mg/m³, ppm)
> ※単位を揃えることに注意！（ppm は $10^{-6}$）

**換気回数 $N$ (回/h)**
$$N = \frac{Q}{V}$$
* $V$: 室容積 (m³)

### 3-2. 音・振動
**騒音レベルの合成 (簡易法)**
* 差が 0〜1 dB → 大きい方に **+3 dB**
* 差が 2〜3 dB → 大きい方に **+2 dB**
* 差が 4〜9 dB → 大きい方に **+1 dB**
* 差が 10 dB以上 → **大きい方のまま**

**距離減衰**
* 点音源：距離2倍で **-6 dB**
* 線音源：距離2倍で **-3 dB**

### 3-3. 熱・流体
**送風機・ポンプの法則 (回転数比 $k$ との変化)**
* 風量 $Q \propto k$ (1乗)
* 圧力 $P \propto k^2$ (2乗)
* 動力 $L \propto k^3$ (**3乗**) !!!最重要!!!

**熱量 $H$ (kJ)**
$$H = c \cdot m \cdot \Delta t$$
* $c$: 比熱 (kJ/(kg·K)) (水は4.186, 空気は約1.0)
* $m$: 質量 (kg)
* $\Delta t$: 温度差 (K)

## 4. 清掃・防除：材質と薬剤の相性
「やってはいけない組み合わせ」を覚える。

| 対象材質 | 弱点・NG洗剤 | 推奨メンテナンス |
| :--- | :--- | :--- |
| **テラゾー** (人造石) | **酸性** (セメント成分が溶ける) | 中性洗剤 |
| **大理石** | **酸性** (炭酸Caが溶ける) | 中性洗剤、空拭き |
| **ゴムタイル** | **溶剤** (溶ける、膨潤する) | 水性ワックス |
| **リノリウム** | **アルカリ性** (黄変する) | 中性洗剤、シール剤必須 |
| **アルミ** | 酸性・アルカリ性 (両性金属) | 中性洗剤、アルマイト処理 |

| 害虫・殺虫剤 | 特徴・キーワード |
| :--- | :--- |
| **ピレスロイド** | **速効性** (ノックダウン)、安全性高い |
| **有機リン剤** | 残効性あり、毒性やや強い |
| **IGR剤** (昆虫成長制御剤) | **遅効性**、脱皮阻害、幼虫に効く |
| **ドブネズミ** | 獰猛、湿地、泳ぐ |
| **クマネズミ** | 警戒心強い、高所、**スーパーラット** |
